# Install

Use either of these repositories:

```
repositories {
    jcenter()
}
```

Or:

```
repositories {
    maven {
        url 'https://dl.bintray.com/novacrypto/Electrum/'
    }
}
```

Add dependency:

```
dependencies {
    compile 'io.github.novacrypto:ElectrumClientRx:0.0.1@jar'
}

```
